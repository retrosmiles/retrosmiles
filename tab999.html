Credits

This is a collection of old smiles mostly used in persian forums.

Special thanks to AsefSoft (QuickSmiles), Worldofugly.de, Pic4Ever.com, yoursmiles.org, BlackMice.com and PersianTools.com members for collecting smiles.

This collection is free to use. If you have any complaints, please contact us here: https://github.com/retrosmiles/retrosmiles/issues/


How To Use

Click on the smiley to copy BBCode to clipboard then go to forum's editor and Paste code there.

Or install and use Extensions.

Note: your forum should support BBCode.


Addition Request

If you want to add more smiles, fork us, add smiles and edit html files (you can use droplet.html to generate html codes quickly), and then send your pull request here: https://github.com/retrosmiles/retrosmiles/pulls
